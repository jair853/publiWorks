Atributos de HTML

En esta página encontrarás algunos de los atributos más comunes en el desarrollo de páginas web a través de HTML (Hypertext Markup Language)
Atributo Elemento(s) al que pertenece Descripción
alt <area>, <img>, <input> Proporciona un texto alternativo cuando el elemento falla en la carga o no se muestra adecuadamente.
autocomplete <form>, <input> Especifica si el elemento formulario o el elemento cuadro de entrada tienen habilitada la opción de autocompletado.
autofocus <button>, <input>, <select>, <textarea> Determina si el elemento adquiere el foco de manera automática cuando la página carga.
autoplay <audio>, <video> Establece que el audio o video se reproduce automáticamente en cuanto carga completamente.
charset <input> Especifica la codificación de los caracteres aceptados por el sitio web.
cols <textarea> Especifica el ancho de las columnas de un área de texto.
colspan <td>, <th> Determina el número de columnas que una celda de una tabla puede ocupar.
for <label>, <output> Determina a cual elemento de un formulario apunta una etiqueta o cálculo en el que se desea mostrar un resultados.
form <button>, <fieldset>, <input>, <label>, <meter>, <object>, <output>, <select>, <textarea> Determina el nombre del o los elementos a quien pertenece.
headers <td>, <th> Establece uno más encabezados a la que la celda pertenece.
