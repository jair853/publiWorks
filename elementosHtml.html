Elementos de HTML

En esta página encontrarás algunos de los elementos utilizados en el desarrollo de páginas web a través de HTML (Hypertext Markup Language)

  • <a>: Se utiliza para crear enlaces dentro y fuera del sitio Web. ( Los enlaces son los que hacen que la internet sea internet, ya que te permiten conectar con diferente contenido!)

  • <address>: Este elemento proporciona información de contacto acerca del autor o dueño del sitio web.

  • <area>: Con esta etiqueta se define el área dentro de un mapa.

  • <audio>: Este elemento permite introducir/anidar archivos de audio dentro de una págna web.

  • <b>: Le da formato de negrita al texto.

  • <blockquote>: Define una sección que es una cita de otra fuente.

  • <body>: Contiene el cuerpo del documento.

  • <br>: Establece un salto de línea en el texto.

  • <button>: Crea un botón para ejecutar una acción específica.

  • <caption>: Introduce una etiqueta o título para una tabla.

  • <dd>: Proporciona la descripción de un término en una lista.br>

  • <div>: Establece un contenedor o sección en un documento.

  • <dl>: Define una lista de descripción.

  • <dt>: Proporciona un nombre o término dentro de una lista de descripción.

  • <em>: Le da énfasis a un texto.

  • <footer>: Establece un pie de página para el documento.

  • <h1> to <h6>: Define el estilo de títulos y subtítulos en el documento.

  • <head>: Contiene metadatos o información acerca del documento o página.

  • <header>: Proporciona un encabezado de un documento o de una sección del documento.

  • <hr>: Introduce una línea o tema que permite diferenciar una sección de otra en el documento.

  • <img>: Permite mostrar una imagen dentro del documento.

  • <li>: Define o introduce un elemento en una lista.

  • <nav>: Permite organizar enlaces de navegación internos o externos.

  • <ol>: Introduce una lista ordenada.

  • <p>: Establece párrafo sencillo en un documento.

  • <section>: Define una sección en un documento.

  • <table>: Establece una tabla para introducir datos.

  • <ul>: Permite estructurar una lista no ordenada.

  • <video>: Permite incluir un video dentro de una página o documento.
